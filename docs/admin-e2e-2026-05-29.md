# Admin Dashboard E2E - 2026-05-29

## Scope

Verifica task AG del 29 maggio: login/admin access, vista statistiche, sospensione utente, verifica `isSospeso`, riabilitazione e verifica finale.

## Flusso verificato

1. Admin autenticato accede agli endpoint protetti `/api/v1/admin/*`.
2. La dashboard legge `/api/v1/admin/statistiche`.
3. Un utente standard viene sospeso con `POST /api/v1/admin/utenti/:id/sospendi`.
4. Il database conferma `isSospeso: true`.
5. Lo stesso utente viene riabilitato con `POST /api/v1/admin/utenti/:id/riabilita`.
6. Il database conferma `isSospeso: false`.
7. La lista admin utenti conferma lo stato finale non sospeso.
